Alors voila les quelques liens que je trouve bien ou qui ont un rapport avec des trucs que je trouve bien. Si vous constatez qu'un lien ne fonctionne pas, soyez gentil de me le dire à BoD@BoDLand.com.

Utils

Recherche

Google : www.google.com le meilleur moteur de recherche.
Altavista : www.altavista.com l'autre moteur de recherche.
Yahoo : www.yahoo.com le meilleur moteur de recherche indexée par thèmes.
Fast ftp search : download.lycos.com/static/advanced_search.asp le meilleur moteur de recherche ftp
RealNames : www.realnames.com pour trouver le site officiel d'une marque.
Asta la vista : astalavista.box.sk le moteur de recherche des cracks, serial, warez, etc.

Dictionnaires

What is : www.whatis.com pour obtenir la définition et des informations sur les termes techniques de l'informatique.
Foldoc : foldoc.doc.ic.ac.uk/foldoc comme whatis.com, en beaucoup plus complet (mais moins beau).
The Jargon Files : www.tuxedo.org/jargon le dictionnaire du jargon informatique, et des hackers.
Dictionnaire des synonymes : elsap1.unicaen.fr/dicosyn.html un dico de synonymes français.
Hachette francophone : www.francophonie.hachette-livre.fr le dictionnaire francophone on-line.
Altavista translation : babelfish.altavista.com les traductions d'altavista en 6 langues.
Voila traduction : www.voila.fr/Trans les traductions de voila : allemand, anglais et francais.
Babylon translation : www.babylon.com/trans Babylon ne traduit que les mots, mais propose plusieurs possibilités suivant le sens.

Bases de données

Internet Movie Database : www.imdb.com la base de donnée des films.
Les pages jaunes : www.pagesjaunes.fr pour avoir les numéros de téléphone et adresses des gens qui sont abonnés à fft.
Infobel : www2.eu-info.com/France un annuaire inversé ! Gratos !
CDDB : www.cddb.com une base de données compilant les titres des pistes de tous les cd audios.

Divers

HoaxBuster : www.hoaxbuster.com première ressource francophone sur les hoax (les canulars qu'on reçoit par mail).
Les changes (Les Echos) : www.lesechos.fr/francais/bourse/changes.htm les changes de la pluspart des monnaies.
Calculator : www.calculator.com le site où on peut tout calculer.
Télérama : www.telerama.fr tous les programmes télé.
Tucows : www.tucows.com le site des logiciels pour le net sous windows.
Allociné : www.allocine.fr pour avoir tous les horraires, toutes les salles, les infos sur les films.
Degriftour : www.degriftour.fr pour voyager pas cher.

Musique

Artistes / groupes

Björk : www.bjork.com Björk Websense, le site officiel.
Jamiroquai : www.jamiroquai.co.uk le site officiel.
Jamiroquai : www.jamiroquai.com le site des fans.
Michael Jackson : www.mjnet.com le site officiel.
Alanis Morissette : www.alanis.com le site officiel.
Tears for fears : www.sonymusic.com/artists/TearsForFears Raoul and the kings of spain, le site officiel.
Sarah McLachlan : www.sarahmclachlan.com le site officiel.
NTM : www.supreme-ntm.com le site officiel.
Tori Amos : www.tori.com le site officiel.

Musique electronique

Scene.org : www.scene.org le meilleur endroit pour trouver des gigz de modules et de démos.
Modplug : www.modplug.com le (meilleur) player de modules sous windows.
Open Cubic Player : www.cubic.org/player/opencp.html le meilleur player de modules (sous DOS).
Monotonik : www.mono211.com un excellent label (hip-hop / electronica / techno et plus).
Maz sound page : www.maz-sound.com tout sur la musique assistée par ordinateur (mod, mid et autres).

MP3

mp3.com : www.mp3.com tout sur le mp3 !!
Winamp : www.winamp.com le (meilleur) player de mp3 sous windows.
Sonique player : www.sonique.com un autre player de mp3 sous windows, assez joli.

Paroles

lyrics.ch : www.lyrics.ch une base de données contenant les paroles de pratiquement toutes les chansons (actuellement inutilisable pour cause de copyrights de merde).
lyrics dot : lyrics.lipetsk.ru celui qui pourrait bien remplacer lyrics.ch, étant en Russie il n'a pas de problèmes de copyrights :)
Singit : www.glop.org/singit le plugin winamp qui joue les paroles façon karaoké.
Kiss this guy : www.kissthisguy.com un site qui ressence des paroles de chansons mal comprises. Très drôle !
Lyrics with time tags : www.come.to/mp3lyrics une base de données contenant des paroles avec des balises de synchronisation (à jouer avec SingIT).

Streaming

RealPlayer : www.real.com le player pour écouter du live sur le net.
Shoutcast : www.shoutcast.com faites votre propre radio sur le net grâce à Shoutcast !
Icecast : icecast.linuxpower.org un concurrent de Shoutcast : il est compatible avec celui-ci mais propose plus d'options, est plus ouvert et libre.
Radios en direct : www.comfm.fr/sites/rdirect une base de données des radios en direct sur le net, classées par thèmes ou par pays.
CKOI live : www.showbizz.net/ckoi.asx la radio québécoise CKOI en direct.

Informatique

Java : java.sun.com la page officielle du java.
Internet Engineering Task Force : www.ietf.org les gens qui font les RFC !
OpenBSD Man pages : www.openbsd.org/cgi-bin/man.cgi les man de OpenBSD on-line.
Wotsit : www.wotsit.org la plus grande collection de formats de fichiers avec leur description.
International Obfuscated C Code Contest : www.ioccc.org le concours de programmation illisible en C : vraiment très drôle et instructif !
Delphi super page : sunsite.icm.edu.pl/delphi tout sur le delphi, en particulier des milliers de composants.
How to write unmaintable code : mindprod.com/unmain.html une page montrant comment programmer pour produire du code impossible à maintenir... Hilarant !
Unicode charts : www.unicode.org/charts les tables de caractères Unicode.

Divers

Publivore : www.publivore.com tout sur les pub tv.
Apple : www.apple.com le site d'Apple. They rulez.
Wired : www.wired.com le magazine wired online.
Slashdot : www.slashdot.org un site de news pour "geeks/nerds" (passionés d'informatique, de technologie, de science) mis à jour plusieurs fois par jour.
Betanews : www.betanews.com les dernières news de dernière minute sur les dernières versions des derniers logiciels !
La page de #nous : hello.to/nous la page du channel IRC #nous.
SETI@home : setiathome.ssl.berkeley.edu le programme américain de recherche des extra-terrestre auquel vous pouvez participer grâce à votre PC.
distributed.net : www.distributed.net partagez un peu de vos ressources de calcul, prouvez que l'union fait la force.
Maurice radio libre : www.club-internet.fr/maurice le site officiel de Maurice.
Arte : www.arte-tv.com la meilleur télé.
GNU : www.gnu.org l'organisation pour les logiciels libres.
Free BSD : www.freebsd.org sans doute le meilleur OS libre.
Perdu : www.perdu.com si vous êtes perdu.
Visual IRC : www.visualirc.com le meilleur client irc, mais n'est plus developpé.
TdC HiD3 0uT : www.multimania.com/azerty0/tdc.html un site parodique sur les "cyber trous-du-cul" (très drôle et tellement vrai).
Silicon creatures : www.micro.magnet.fsu.edu/creatures de minuscules creatures se cachent dans nos processeurs, découvrez quelques clichés ici.
Fonts 'n things : www.fontsnthings.com ce site offre des polices gratuites.
Random link : random.yahoo.com/bin/ryl un lien qui envoie sur un site au hasard (un service yahoo.com).
