Le Java est un langage de programmation inventé par Sun. C'est un langage objet compilé interpreté multiplateformes ! Autrement dit, un programme Java peut tourner sur tout système (à condition que la "Java virtual machine" existe sur ce système). De plus, de nombreux outils sont disponibles, notamment pour le web avec les applet. Les applet sont des programmes Java faits pour être éxécutés à l'intérieur d'une page web. Bien que n'étant pas encore le langage prédominant, de nombreux développeurs lui portent un très grand interêt et d'apres moi, le Java est le langage du futur !! Et JavaOS quand il existera, le système d'exploitation du futur !
Enfin... en tout cas, moi j'ai fait deux applets ridicules en Java : TTSB et Jibbles. J'ai aussi deux projets en préparation, ils sont presque finis mais il reste des bugs. Il s'agit d'un serveur web et un serveur de news. Il seront disponibles (peut-être) très prochainement ! Stay tuned :) En ce moment, je fais aussi un bot IRC, et d'autres trucs ; pour des infos "up-to-date", consultez mon finger.
Et pour des infos sur le java, allez sur la page du java : http://java.sun.com.
